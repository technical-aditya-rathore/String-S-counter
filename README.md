
 String Substring Counter (Java)
 ---------------------------------------------------------------------------------------------------------------------
 Description

This Java console application checks how many times a substring occurs inside a given string.
The program allows user input, processes the string using core Java logic, and prints the total number of matches found.

JUnit/TestNG tests are included to verify correctness.

 Features
 ----------------------------------------------------------------------------------------------------------------------

Accepts string and substring from user input

Counts non-overlapping substring occurrences

Displays message if substring is not found

Modular static methods for clean logic

Unit testing using TestNG + JUnit

 Core Logic
 -----------------------------------------------------------------------------------------------------------------------

Iterates through the main string

Uses substring() comparison

Skips already matched segments to avoid overlap
